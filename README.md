# Aulas de Programação para Web

Projeto com atividades e layouts de interface para estudos de HTML, CSS e JavaScript.

## Destaque atual

- Layout visual com tema claro/escuro
- Tela de carregamento animada
- Organização em pastas:
  - `1°trabalho/cara-car.html`
  - `1°trabalho/css/style.css`
  - `1°trabalho/js/script.js`

## Como executar

1. Abra `1°trabalho/2048.html` no navegador.
2. Use o botão de tema para alternar entre claro e escuro.
